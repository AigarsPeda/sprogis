# Sprogis

A WordPress-based project for semantic personal publishing and travel listings.

## Project Overview
This project is built on top of WordPress, providing a robust platform for content management and extensibility. It includes custom plugins and themes, such as the `travel-listings` plugin, to enhance functionality for travel-related content.

## Installation
1. Clone this repository:
   ```sh
   git clone https://github.com/AigarsPeda/sprogis.git
   ```
2. Place the files in your web server's public directory.
3. Create a database and configure your `wp-config.php` file with the correct database credentials.
4. In your browser, open `wp-admin/install.php` and follow the on-screen instructions to complete the WordPress setup.

## Requirements
- PHP 7.2.24 or greater (recommended: 8.3+)
- MySQL 5.5.5 or greater (recommended: MySQL 8.0+ or MariaDB 10.6+)
- Apache with `mod_rewrite` enabled
- HTTPS support recommended

## Updating
- Use the WordPress admin dashboard for automatic updates.
- For manual updates, back up your files and database, replace old files with new ones, and run `wp-admin/upgrade.php`.
